# MOBILE-IDE-Main 🤖🔍

Groq API ကို အသုံးပြုပြီး တည်ဆောက်ထားသည့် Android AI Search & Code IDE Application ဖြစ်ပါသည်။ AI ထံမှ တောင်းဆိုလိုက်သည့် HTML/CSS/JS ကုဒ်များကို App အတွင်း Live Preview Window ဖြင့် ချက်ချင်း Run ပြသပေးနိုင်ပါသည်။

---

## 🚀 Features (ပါဝင်သော လုပ်ဆောင်ချက်များ)

* **Groq AI Integration:** Groq API Key ထည့်သွင်းပြီး မြန်ဆန်သော AI Chatbot အဖြစ် အသုံးပြုနိုင်ခြင်း။
* **Live Web Preview:** AI မှ ရေးသားပေးသော Web Code (HTML) များကို WebView ဖြင့် တိုက်ရိုက် Dynamic မောင်းနှင်ပြသပေးခြင်း။
* **Floating Preview Dialog:** UI ဒီဇိုင်းများကို Dialog Window ဖြင့် သီးသန့် Pop-out လုပ်၍ စမ်းသပ်နိုင်ခြင်း။
* **Chat History Management:** အရင်ပြောဖူးသမျှ Conversation Session များကို သိမ်းဆည်းပေးထားပြီး ပြန်လည်ဖတ်ရှုနိုင်ခြင်း။

---

## 🛠️ Built With (အသုံးပြုထားသည့် နည်းပညာများ)

* **Language:** Kotlin
* **UI Layout:** Android XML Layouts
* **Target SDK:** Android (Activity-based Framework)
* **AI Engine:** Groq API

## ⚙️ Setup & Installation (စတင်အသုံးပြုပုံ)

1. ဒီ Repository ကို Clone လုပ်ပါ သို့မဟုတ် Zip ဒေါင်းလုဒ်ဆွဲပါ။
2. Android Studio တွင် Project ကို ဖွင့်ပါ။
3. App ကို Build လုပ်ပြီး Android Device/Emulator တွင် Install တင်ပါ။
4. App အတွင်းရှိ Settings (⚙️) သို့သွားပြီး သင်၏ [Groq API Key](https://console.groq.com/keys) ကို ထည့်သွင်းပါ။
5. "Create a login screen HTML" ဟု ရိုက်နှိပ်၍ စမ်းသပ်ကြည့်နိုင်ပါသည်။

6. # MOBILE-IDE-Main 🤖🔍

An Android AI Search & Code IDE Application built using the Groq API. It allows users to prompt the AI for web code (HTML/CSS/JS) and instantly view the live execution within an integrated Live Preview Window.

---

## 🚀 Features

* **Groq AI Integration:** Input your Groq API Key in settings to enable a high-speed, interactive AI chatbot experience.
* **Live Web Preview:** Automatically parses and executes web code (HTML) generated by the AI using an integrated WebView for real-time dynamic rendering.
* **Floating Preview Dialog:** Pop out your active UI designs into a floating dialog window to test responsiveness and interactivity on the fly.
* **Chat History Management:** Saves past conversation sessions locally, allowing you to seamlessly manage and switch between different chats.

---

## 🛠️ Built With

* **Language:** Kotlin
* **UI Layout:** Android XML Layouts
* **Target SDK:** Android (Activity-based Framework)
* **AI Engine:** Groq API

---

## ⚙️ Setup & Installation

1. Clone this repository or download the source code as a ZIP file.
2. Open the project in Android Studio.
3. Build the app and install it on your Android Device or Emulator.
4. Navigate to Settings (⚙️) inside the app and enter your [Groq API Key](https://console.groq.com/keys).
5. Start experimenting by typing prompts like: `"Create a colorful calculator web app with fully working buttons"`.
